劳斯判据：
一个线性系统稳定的充要条件：闭环特征方程的所有根(极点)，都在复平面的
左平面，且是实部全为负。
做法：
第一步(必要条件)：特征多项式的所有系数必须都存在且挺好(通常要求全为正)。
                有缺项或有负系数，直接判不稳定。
第二步(劳斯表)：把系数排成劳斯表，逐行计算。
对三阶系统：
$$
a_{3}·s^3 + a_{2}·s^2 + a_{1}·s + a_{0}
$$

| 行       | 第一列                                 | 第二列       |
| ------- | ----------------------------------- | --------- |
| $$s^3$$ | $$a_{3}$$                           | $$a_{1}$$ |
| $$s^2$$ | $$a_{2}$$                           | $$a_{0}$$ |
| $$s^1$$ | $$(a_{2}·a_{1}-a_{3}·a_{0})/a_{2}$$ | $$0$$     |
| $$s^0$$ | $$a_{0}$$                           |           |

结论：劳斯表第一列全为正<-->系统稳定
第一列符号改变次数 = 右平面根的个数
