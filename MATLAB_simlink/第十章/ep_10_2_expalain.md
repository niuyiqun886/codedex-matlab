状态空间模型是什么
传递函数只描述"输入→输出"的外部关系，而状态空间模型把系统内部的状态变量 x 也描述出来：


x' = Ax + Bu：状态方程，描述内部状态如何随输入演化
y = Cx + Du：输出方程,描述输出如何由状态和输入组成

四个矩阵的含义（设 n 个状态、m 个输入、p 个输出）：

A (n×n) 系统矩阵：决定系统自身动态，A 的特征值就是系统极点，决定稳定性
B (n×m) 输入矩阵：输入如何影响各状态
C (p×n) 输出矩阵：输出由哪些状态组合而成
D (p×m) 直通矩阵：输入直接到输出的通道，多数物理系统 D=0

例10-2状态空间模型的完整推导过程
第一步：写出状态方程

状态方程：
x' = Ax + Bu                        ----(1)
y = Cx + Du                         ----(2)

A = [0 1 0; 0 0 1; -1 -2 -3]
B = [0; 0; 1]
C = [1 0 0]
D = 0

带入:
x' = [0 1 0; 0 0 1; -1 -2 -3][x1; x2; x3] + [0; 0; 1]u
逐行展开：得到状态方程
x1' = x2                            ----(3)
x2' = x3                            ----(4)
x3' = -x1 + -2*x2 + -3*x3 + u       ----(5)

输出方程：
y = [1 0 0][x1; x2; x3] + 0·u = x1  ----(6)

思路是将(5)中的都替换为y和u的表达式
x1 = y
x2 = y'
x3 = y''
x3' = y'''
带入式(5)中：得
y''' + 3·y'' + 2·y' + y = u         ----(7)

这个状态空间模型的本质就是一个三阶的微分方程。

第二步：求传递函数，sys2 = tf(sys1)
对(1)两侧进行拉普拉斯变换，其中x是一个3行一列的列向量。
可以得到：(sI - A)·x = B·u 
          x = (sI - A)^-1·B·u       ----(8)
将(8)带入(2)中得：其中：I为单位矩阵
G(s) = C·(sI - A)^-1·B + D          ----(9)


反向操作：将一个三阶微分方程→状态空间的拆法
将式(7)拆为状态空间模型：
第 1 步：定义状态变量 —— 取 y 及其各阶导数
n 阶方程需要 n 个状态。选法是从 y 开始，每求一次导定义一个新变量：
x1 = y
x2 = y'
x3 = y''
第 2 步：写出前 n−1 个方程---纯粹是定义的结果
x1' = y' = x2
x2' = y'' = x3
第 3 步：最后一个方程---用原微分方程解出最高阶导数
y''' = - 3·y'' - 2·y' - y + u
将上式全补换为状态变量：
x3' = y''' = - x1 - 2·x2 - 3·x3 + u

最后：将x1', x2', x3' 组合
x1' =          x2
x2' =                 x3
x3' = - x1 - 2·x2 - 3·x3 + u
得：
[x1'; x2'; x3'] = x' = [0 1 0; 0 0 1; -1 -2 -3][x1; x2; x3] + [0; 0; 1]u
