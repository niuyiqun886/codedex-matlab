# 2.4 MATLAB 的程序流控制
## 2.4.1 循环控制结构

for循环结构：
```
for 循环变量 = 向量表达式
	循环体语句
end

for i = 1 : 1 : 20 (从1到20，间隔1)

end
```

while循环结构：
```
while 关系表达式
	循环体语句组
end

while i < 20

end
```

添加写操作语句：

fprintf用法：
```
fprintf('字符串', 变量)
```

常用格式占位符：

|  占位符  |     含义      |
| :---: | :---------: |
|  %d   |     整数      |
|  %f   | 浮点数（默认6位小数） |
| % .2f |  浮点数保留两位小数  |
|  %e   |    科学计数法    |
|  %s   |     字符串     |

disp：直接输出变量值，里面写什么就输出什么

input的用法(和python一样)：
```
变量 = input('提示信息');          % 接收数字或表达式
变量 = input('提示信息', 's');     % 接收字符串
```
